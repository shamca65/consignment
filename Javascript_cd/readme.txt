README file
for the CD-ROM that accompanies
"The JavaScript Annotated Archive"

+++++++++++++++++++++++++++++++++++

Each JavaScript example from the book is represented on the
CD-ROM.

To access and browse the CD-ROM, you must load the file
INDEX.HTM into your Web browser.

From there, you will see all the scripts arranged by chapter,
as organized in the book.
